# Description du jeu Super Pac-Man

Dans ce jeu, le joueur joue en tant qu'un Pac-Man, une pastille jaune qui aime manger. Il existe trois niveaux, avec la difficulté qui augmente a chaque niveau. Le but du jeu est de passer la dernière Gate du dernier niveau avec le score le plus haut possible. Vous commencez avec 3 points de vie et vous perdez un a chaque fois que vous êtes mort.

#Controles
Les seules contrôles du jeu sont les quatre flèches du clavier, utilisées pour orienter le Pac-Man dans la direction voulue.

##Obstacles

Certains obstacles vont essayer de vous empêcher	de gagner. Le premier consiste des fantômes. Ces derniers vous tuent si vous vous trouvez dans la meme cellule avec eux. Ils existent sous plusieurs formes: Blinky, Inky et Pinky. Inky et Pinky sont plus dangereux parce qu'ils vont tracker le Pac-Man pour le tuer.
Le deuxième obstacle consiste des saw blades. Ces derniers ont la forme d'un carre avec un petit carre vide dedans. Ils bougent soit verticalement ou horizontalement et ils vous tuent si vous vous trouvez dans la meme cellule avec eux.
Le troisième consiste des Turrets. Ces derniers envoient des projectiles mortels dans une seule direction.

## Affronter les fantômes
Meme si les fantômes vous tuent, vous pouvez manger des pieces qui se situent dans le jeu pour effrayer les fantômes pour une période de 10 secondes. Durant cette période les fantômes seront d'une couleur bleue et c'est le Pac-Man qui peut manger les fantômes pendant cette période.

##Passer d'un niveau a l'autre

Pour passer au niveau suivant vous devez faire disparaître le Gate qui vous bloque. Pour cela vous devez manger les clés correspondantes et pour les niveaux 2 et 3 il faut aussi manger tous les diamants du niveau.

## Obtenir du score
Chaque diamant mangé vous ajoute 10 points au score, chaque cerise mangée vous ajoute 200 points au score et chaque fantôme effrayé mangé vous ajoute 500 points au score.

## Obtenir des points de vie
Pour obtenir des points de vie, vous pouvez manger des coeurs qui se trouvent dans les niveaux 2 et 3, chaque coeur mangé vous ajoute 1 point de vie.
